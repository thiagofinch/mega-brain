## Como Usar o Nome da Empresa

Ao gerar logs reais a partir deste template:
1. Ler `READ_AI_COMPANY_NAME` do `.env` (ex: "Acme Corp")
2. Extrair abreviação inteligente: primeiro nome ou sigla (ex: "ACME")
3. Substituir `{COMPANY_SHORT_NAME}` no header ASCII pelo nome abreviado
4. A cor vermelha (🔴) identifica visualmente o Bucket 2 (Business)

**Regra de abreviação:**
- Nome curto (1 palavra): usar completo → "Acme" → `ACME`
- Nome longo (2+ palavras): usar primeiro nome → "Mega Brain AI" → `MEGA`
- Sigla oficial: usar sigla → "XPTO Consulting Group" → `XPTO`
