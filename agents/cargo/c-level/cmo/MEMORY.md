├─────────────────────────────────────────────────────────────────────┤

│  → Copy: Ofertas, urgência, objeções                               │
│                                                                     │
│  Level 5: MOST AWARE                                                │
│  → Já conhece e confia, pronto para comprar                        │
│  → Copy: Deal, preço, CTA direto                                   │
│                                                                     │
│  INSIGHT: Match your message to awareness level                     │
