**ROIC Calculation Framework (Josh Troy - BATCH-073):**
```
┌─────────────────────────────────────────────────────────────────────┐
│  ROIC - RETURN ON INVESTED CAPITAL                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  Formula: (Profit - Investment) / Investment × 100                  │
│                                                                     │
│  EXAMPLE:                                                           │
│  ├─ Investment: $42,000                                            │
│  ├─ Revenue Generated: $222,000                                    │
│  ├─ Profit: $180,000                                               │
│  └─ ROIC: ($180K - $42K) / $42K = 328% → ~400% ROIC                │
│                                                                     │
│  BENCHMARKS:                                                        │
│  ├─ 100% ROIC = "okay"                                             │
│  ├─ 200-300% ROIC = "good"                                         │
│  ├─ 400%+ ROIC = "excellent"                                       │
│  └─ 900% ROIC = Josh Troy's company                                │
│                                                                     │
│  PRE-TAX PHILOSOPHY:                                                │
│  ├─ $1 invested pre-tax = $1 working                               │
│  ├─ $1 invested post-distribution = $0.60 working                  │
│  └─ Reinvest profits BEFORE taking distributions                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
