### METODOLOGIAS DE ESCALA

#### 4-Week Firing Protocol [BATCH-069]
```
Semana 1: Identificar underperformance
Semana 2: Coaching intensivo + metas claras
Semana 3: Monitoramento diário + feedback
Semana 4: Decisão final (melhorou ou sai)

→ NÃO estender além de 4 semanas
→ Manter underperformer = punir top performers
```

#### Conviction Level Dynamics [BATCH-069]
```
Antes da call: Prospect chega com 3-4/10
Durante discovery: Subir para 6/10 (mínimo para pitch)
Durante pitch: Subir para 8/10
No close: Precisa estar 9+/10

Se não chegar em 6/10 no discovery → não fazer pitch
```

#### Price First Strategy [BATCH-069]
```
Revelar range de preço nos primeiros 10 minutos:

"Só para você ter uma ideia, nossos programas
ficam entre X e Y dependendo do que faz sentido.
Isso está dentro do que você estava pensando?"

→ Elimina tire-kickers cedo
→ Reduz objeção de preço no final em 30-40%
→ Mostra confiança no valor
```

#### Value-Driven Follow-up [BATCH-068]
```
Cada follow-up deve agregar valor:
1. Case study relevante
2. Recurso útil (template, checklist)
3. Insight de mercado
4. Resultado de cliente similar

NUNCA: "Só checando se viu meu email"
```

### MODELOS MENTAIS DE REVENUE

#### Scarcity vs Abundance Mindset [BATCH-127]
```
SCARCITY (errado):
- Aceita qualquer cliente
- Reduz preço para fechar
- Medo de perder deal

ABUNDANCE (certo):
- Qualifica agressivamente
- Mantém preço firme
- Confiança de que há mais leads
```

#### Exit Readiness Checklist [BATCH-083]
```
Para ser "buyable" a empresa precisa:
[ ] $2M+ EBITDA mínimo
[ ] Diversificação de clientes (nenhum >25%)
[ ] Sistemas documentados
[ ] Time que opera sem founder
[ ] Receita recorrente ou previsível
[ ] Clean financials (3+ anos)
```

#### Cleaner Role Model [BATCH-068]
```
Estrutura de 2 closers:
- Closer Principal: 80-85% das calls
- Cleaner: 15-20% das calls difíceis

Cleaner atua em:
- Deals >$25K
- Prospects sofisticados
- Casos complexos
- Recovery de negociações travadas
```
