### NEPQ Precision Probing - 4 Questions Framework
```
┌─────────────────────────────────────────────────────────────────────────────┐
│  NEPQ PRECISION PROBING - QUANDO USAR: Building the Gap                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  Q1: REPETIÇÃO DE PALAVRA EMOCIONAL                                        │
│  └── Prospect: "It's causing me so much stress..."                         │
│  └── Você: "...stress?" [concerned tone, hand on chest]                    │

│  └── "Has that had a... has that had an impact on you?"                    │
│  └── "What's that doing to you?" [concerned tone]                          │
```
