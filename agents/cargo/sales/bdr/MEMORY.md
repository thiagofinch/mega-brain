## SCRIPTS ADAPTADOS BR

### Cold Call Opening
```
"Oi [Nome], aqui é [Meu Nome] da [Empresa].

Estou ligando porque ajudamos [tipo de empresa] a
[resultado principal] e vi que vocês [contexto].

Você tem 2 minutinhos?"

[Se SIM] → Discovery rápida
[Se NÃO] → "Entendo, qual melhor horário para uma conversa de 5 min?"
```

### WhatsApp First Touch
```
"Oi [Nome], tudo bem?

Aqui é [Meu Nome] da [Empresa].

Vi [contexto/conexão] e pensei que faria sentido
uma conversa rápida sobre [benefício principal].

Você teria 5 min essa semana?"
```

### Objeção "Não tenho tempo"
```
"Entendo completamente. Justamente por isso que
queria uma conversa rápida - para ver se faz
sentido você investir mais tempo.

5 minutinhos agora ou prefere que eu retorne
[dia/horário]?"
```
