# Inbox

Adicione seus materiais de especialistas aqui.

## Tipos de conteúdo aceitos
- URLs de YouTube (transcrições serão extraídas automaticamente)
- Arquivos PDF
- Transcrições em .txt ou .md
- Documentos estratégicos

## Como usar
Execute `/ingest [URL ou caminho]` no Claude Code para processar o material.

## Organização
Os arquivos serão organizados automaticamente pelo pipeline JARVIS em subpastas por fonte/especialista.
