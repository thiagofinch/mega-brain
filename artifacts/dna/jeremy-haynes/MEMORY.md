## METODOLOGIAS DISPONÍVEIS

### MET001: Hammer Them Strategy (Full Process)

**Quando usar:** Qualquer funnel high-ticket com call de vendas

**Pré-requisitos:**
- Content library pronta (30-50 short, 15-30 long)
- Retargeting pixel instalado
- Sales team treinado em outreach

**Etapas:**

```
┌─────────────────────────────────────────────────────────────────────┐
│ HAMMER THEM - Timeline de Execução                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  Day -30 to -7:                                                     │
│  └─ Build content library (30-50 short, 15-30 long)                │
│     ↓                                                               │
│  Day -3:                                                            │
│  └─ Start heavy retargeting of booked leads                        │
│     ↓                                                               │
│  Day -2:                                                            │
│  └─ Intensify with short form + long form mix                      │
│     ↓                                                               │
│  Day -1:                                                            │
│  └─ Sales team manual outreach begins                              │
│     ↓                                                               │
│  Day 0 (CALL):                                                      │
│  └─ 15-20+ touchpoints achieved, lead arrives pre-sold             │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**Métricas de sucesso:**
- Show rate > 70%
- Close rate > 30%
- Rep feels like "cashier"

**Fonte:** ^[JH-SOP002_001, JH-SOP002_002, JH-SOP002_010, JH-SOP002_012]

### MET002: Challenge Funnel (4-Day Structure)

**Quando usar:** Lançamentos, construção de autoridade, ofertas $10k+

**Pré-requisitos:**
- Backend offer definida ($15-20k típico)
- Ticket pricing decidido ($1-$7 comprovado)
- Social proof documentado

**Etapas:**

```
┌─────────────────────────────────────────────────────────────────────┐
│ CHALLENGE FUNNEL - 4 Days                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  DAY 1: HOOK + PROBLEM                                              │
│  ├─ Hook that connects with pain                                    │
│  ├─ Problem identification                                          │
│  └─ Initial value delivery                                          │
│                                                                     │
│  DAY 2: PROOF + VULNERABILITY                                       │
│  ├─ Multi-layered social proof (30-45 min)                         │
│  │   ├─ Client results                                              │
│  │   ├─ Industry recognition                                        │
│  │   └─ Expertise demonstration                                     │
│  └─ Vulnerability/Authenticity (15-20 min)                         │
│      └─ Human connection, relatability                              │
│                                                                     │
│  DAY 3: SOLUTION REVEAL                                             │
│  ├─ Framework teaching                                              │
│  ├─ "How" (not just "what")                                        │
│  └─ Tangible takeaways                                              │
│                                                                     │
│  DAY 4: CLOSE                                                       │
│  ├─ Offer presentation                                              │
│  ├─ Phone CTA (not just buy button)                                │
│  └─ Urgency/scarcity if applicable                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**Benchmarks comprovados:**
- $7 tickets × 717 sold + $15k backend ✓
- $1 tickets × 293 sold → $119k collected ✓

**Fonte:** ^[JH-SOP024_011, JH-SOP024_021, JH-SOP024_031, JH-SOP024_041]

### MET004: Cold Video Pitch Process

**Quando usar:** Outbound B2B, agency prospecting

**Etapas:**
```
┌─────────────────────────────────────────────────────────────────────┐
│ COLD VIDEO PITCH - Step by Step                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  STEP 1: PREPARE                                                    │
│  └─ Lead list ready in batch (DON'T hunt between pitches)          │
│                                                                     │
│  STEP 2: RECORD                                                     │
│  └─ 5-15 min personalized video per lead                           │
│                                                                     │
│  STEP 3: ADDRESS                                                    │
│  └─ Common questions/objections proactively                        │
│                                                                     │
│  STEP 4: SEND                                                       │
│  └─ Via email or DM                                                 │
│                                                                     │
│  STEP 5: FOLLOW UP                                                  │
│  └─ Push to call booking                                            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**Benefício:** Shortens sales cycle by reducing questions/objections upfront

**Fonte:** ^[JH-SOP021_001, JH-SOP021_002]
