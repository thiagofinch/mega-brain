# Logs

Logs de sessões e processamento. Gerados automaticamente.

## Subpastas
- `sessions/` - Logs de sessões do JARVIS
- `batches/` - Logs de processamento em batch
- `decisions/` - Registro de decisões entre agentes
- `reports/` - Relatórios gerados

Estes logs são locais e não são rastreados pelo git.
