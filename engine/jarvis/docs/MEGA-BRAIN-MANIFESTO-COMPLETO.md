#### Etapa 4.1: Chunking Semântico

O texto bruto é quebrado em unidades semânticas - não por parágrafos ou páginas, mas por **significado**.

```
TEXTO ORIGINAL (fragmento):
"...e o segredo do follow-up é que você não pode parar no terceiro
contato. A maioria para. Mas os dados mostram que 80% das vendas
acontecem entre o quinto e décimo segundo contato. Então quando
você constrói uma sequência de 12 toques, você está onde a maioria
das vendas realmente acontece..."

                              ▼

CHUNK GERADO:
{
  "chunk_id": "JH_042_003",
  "source_file": "[JH-0042] Follow-Up Mastery.txt",
  "content": "80% das vendas acontecem entre 5º e 12º contato...",
  "semantic_type": "heuristica_com_numero",
  "domain": "follow-up",
  "confidence": 0.92
}
```

**Resultado: 2.800+ chunks semânticos** - cada um é uma unidade atômica de conhecimento.
