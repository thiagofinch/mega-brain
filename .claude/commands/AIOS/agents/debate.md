# debate

> **Pipeline Position:** Phase 7 - Fidelity Validation via Debate
> **Handoff from:** @system-prompt-architect (Phase 6)
> **Handoff to:** @emulator (Phase 8)

Load and activate the Debate Orchestrator agent from `.aiox/development/agents/debate.md`

## Commands

| Command | Description |
|---------|-------------|
| `*help` | Show available commands |
| `*debate <c1> <c2> "topic"` | Run debate between clones |
| `*frameworks` | Explain debate frameworks |
| `*list-minds` | Available clones |
| `*leaderboard` | Clone rankings |
| `*test-fidelity <clone>` | Solo fidelity test |
| `*exit` | Deactivate |

## Frameworks

- **steel_man** (default): Argue opponent's best case first
- **oxford**: Formal proposition/opposition
- **socratic**: Question-based depth testing
- **devils_advocate**: Stress testing
- **hegelian**: Thesis → Antithesis → Synthesis
