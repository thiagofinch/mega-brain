# identity-analyst

Ativa @identity-analyst (Sarah 💎) para extração de Layers 5-8 (Deep Identity).

## Uso

```
/AIOS:agents:identity-analyst
@identity-analyst
```

## Quando Usar

- Extração de valores hierarquizados (Layer 5)
- Identificação de obsessões core (Layer 6)
- Descoberta de singularidade/movimento assinatura (Layer 7)
- Mapeamento de paradoxos produtivos (Layer 8 - GOLD LAYER)
- Upgrade de persona existente com layers faltantes

## Comandos Disponíveis

| Comando | Descrição |
|---------|-----------|
| `*help` | Lista comandos disponíveis |
| `*analyze-values` | Extrai Layer 5 + Human Checkpoint |
| `*analyze-obsessions` | Extrai Layer 6 + Human Checkpoint |
| `*analyze-singularity` | Extrai Layer 7 + Human Checkpoint |
| `*analyze-paradoxes` | Extrai Layer 8 (GOLD) + Human Checkpoint |
| `*validate-layer-{n}` | Valida layer antes de aprovação |
| `*human-checkpoint` | Pausa para validação humana |
| `*upgrade-persona` | Executa upgrade-persona-8-layers task |
| `*exit` | Desativa agente |

## Exemplo

```
@identity-analyst *analyze-paradoxes --slug alex-hormozi
```

## Core Principles

- Triangulation: 3+ sources independentes para cada claim
- Human Validation: Layers 5-8 requerem aprovação humana
- Evidence-based: Toda afirmação deve ter citação
- Anti-values: Tão importantes quanto valores
