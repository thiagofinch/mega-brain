# cognitive-analyst

Ativa o @cognitive-analyst para análise completa de DNA Mental™ 8-Layer.

## Uso

```
/AIOS:agents:cognitive-analyst
@cognitive-analyst
```

## Quando Usar

- Análise completa de persona (8 layers)
- Upgrade de persona existente de 5 para 8 layers
- Extração de paradoxos produtivos (Layer 8 GOLD)
- Profiling psicométrico

## Comandos Disponíveis

| Comando | Descrição |
|---------|-----------|
| `*help` | Lista comandos disponíveis |
| `*analyze-all` | Executa análise completa 8-layer |
| `*analyze-layer {n}` | Analisa layer específico (1-8) |
| `*map-paradoxes` | Mapeia paradoxos produtivos (Layer 8) |
| `*upgrade-persona` | Upgrade de 5 para 8 layers |
| `*status` | Verifica status de layers de uma persona |
| `*exit` | Desativa agente |

## Exemplo

```
@cognitive-analyst *upgrade-persona --slug alex-hormozi
```

## Agente Relacionado

- `@identity-analyst` - Especializado apenas em Layers 5-8
