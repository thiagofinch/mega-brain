# SOW: SDR (Sales Development Representative)

> Statement of Work Completo

---

## 📋 METADATA

| Campo | Valor |
|-------|-------|
| ID | S03 |
| Área | SALES |
| Reporta Para | Sales Manager |
| Senioridade | Júnior/Pleno |
| Tipo | Full-time |
| AGENT | AGENT-SDR |

---

## 🎯 MISSÃO

> Qualificar leads e agendar reuniões de alta qualidade para closers, garantindo que apenas prospects com fit e interesse real cheguem na call.

---

## 📊 KPIs

| KPI | Meta | Frequência |
|-----|------|------------|
| Agendamentos | 10-15/semana | Semanal |
| Show rate | ≥ 75% | Semanal |
| Contatos/dia | 50-80 | Diário |
| Taxa qualificação | ≥ 20% | Semanal |
| Tempo resposta | < 5 min | Contínuo |
| Conectividade | ≥ 30% | Semanal |

---

## ✅ RESPONSABILIDADES

### Diárias
| Atividade | Tempo | Output |
|-----------|-------|--------|
| Responder leads novos | Contínuo | < 5 min SLA |
| Ligações de qualificação | 3-4h | 50-80 contatos |
| Follow-up sequências | 1h | Mensagens |
| Agendar no calendário | Contínuo | Agendas |
| Atualizar CRM | 30min | Pipeline limpo |
| Confirmar agendas amanhã | 30min | Confirmações |

### Semanais
- Análise de show rate
- Sync com closers
- 1:1 com gestor
- Ajuste de scripts

---

## ⏰ ROTINA TÍPICA

```
08:00 - 08:30  │ Revisar leads novos, priorizar

09:00 - 12:00  │ BLOCO DE LIGAÇÕES
12:00 - 13:00  │ Almoço
13:00 - 13:30  │ Confirmar agendas de amanhã
13:30 - 16:30  │ BLOCO DE LIGAÇÕES + FOLLOW-UP
16:30 - 17:00  │ Atualizar CRM
17:00 - 17:30  │ Enviar lembretes
17:30 - 18:00  │ Prep dia seguinte
```

## 💬 COMUNICAÇÃO

| Com Quem | Frequência | Assunto |
|----------|------------|---------|
| Closer | Diária | Qualidade leads |
| SM | Diária | Pipeline |
| Marketing | Semanal | Qualidade leads |

### Handoff para Closer
```
1. Qualificar lead (BANT/SPIN)
2. Criar nota no CRM:
   - Resumo da conversa
   - Dor principal
   - Budget/timeline
   - Expectativas
3. Agendar no calendário do closer
4. Notificar no Slack
```

## 📈 PROGRESSÃO

```
[SDR] → Closer
[SDR] → SDR Lead (se 4+ SDRs)
[SDR] → BDR (outbound)
```

### Critérios para Closer
- 6+ meses como SDR
- Show rate > 75% consistente
- Aprovação em role-play
