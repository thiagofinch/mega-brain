# SOW: BDR (Business Development Representative)

---

## 📋 METADATA

| Campo | Valor |
|-------|-------|
| ID | S04 |
| Área | SALES |
| Reporta Para | Sales Manager |
| Senioridade | Pleno |
| AGENT | AGENT-BDR |

---

## 🎯 MISSÃO

> Gerar novas oportunidades através de prospecção ativa (outbound), identificando empresas/pessoas que se beneficiariam mas ainda não nos conhecem.

---

## 📊 KPIs

| KPI | Meta |
|-----|------|
| Contatos/dia | 50-100 |
| Conexões/semana | 20-30 |
| Meetings agendados | 5-10/semana |
| Pipeline gerado | R$ X/mês |
| Taxa de resposta | > 5% |

---

### Diárias
| Atividade | Tempo |
|-----------|-------|
| Pesquisar prospects | 1h |
| Enviar sequências outreach | 2h |
| Ligações prospecção | 3h |
| Follow-up | 1h |
| CRM | 30min |

---

## 🔄 DIFERENÇA SDR vs BDR

| SDR | BDR |
|-----|-----|
| INBOUND | OUTBOUND |
| Lead demonstrou interesse | Lead não conhece |
| Qualificar e agendar | Prospectar e conectar |
| Mais volume | Mais estratégia |
| Mensagens diretas | Personalização alta |

---

## 💰 COMPENSAÇÃO

| Componente | % | Valor BR |
|------------|---|----------|
| Base | 60-70% | R$ 3.000 - R$ 5.000 |
| Variável | 30-40% | Por meeting + pipeline |
| **OTE** | 100% | R$ 5.000 - R$ 9.000 |
