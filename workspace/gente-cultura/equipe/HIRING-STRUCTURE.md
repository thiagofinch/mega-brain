## Workflow de Novo Candidato

1. **Duplicar** o template oficial
2. **Renomear** para `SC-[CARGO]-[NNN]-[NOME]`
3. **Mover** para a subpasta correta em `02-SCORECARDS/`
4. **Preencher** com analise do candidato
5. **Atualizar** `_INDEX.md` com status

## CLI Commands

```bash
# Criar novo scorecard
python3 scripts/gdrive_manager.py scorecard CTO "Nome Candidato"
python3 scripts/gdrive_manager.py scorecard SM "Nome Candidato"
python3 scripts/gdrive_manager.py scorecard CLO "Nome Candidato"

# Listar pasta
python3 scripts/gdrive_manager.py list <folder_id>

# Mover arquivo
python3 scripts/gdrive_manager.py move <file_id> <new_parent_id>

# Renomear arquivo
python3 scripts/gdrive_manager.py rename <file_id> "Novo Nome"
```
