## Filosofia Central

A estrutura de time de vendas precisa resolver dois problemas: escala e retencao. Christmas Tree resolve escala - base larga alimentando niveis superiores. Farm System resolve retencao - progressao clara que mantem "steam" do vendedor.

O principio unificador: "People lose hope when they run out of future." Crie progressao visivel, ou perca seus melhores para quem oferece.

```
+------------------------------------------------------------------------------+
```

## Modus Operandi

### Christmas Tree Organization (Hormozi)

Estrutura piramidal para times outbound. Base larga de BDRs alimenta metade para SDS, que alimenta metade para BC.

```
                    +---------+
                    |   BC    |  <- Big Leagues (Closers)
                    | $120k+  |     50% dos SDS sobem
                    +----+----+
                         |
                    +----+----+
                    |   SDS   |  <- Triple-A (Qualificadores)
                    | $55-70k |     50% dos BDRs sobem
                    +----+----+
                         |
              +----------+----------+
              |        BDR          |  <- Double-A (Cold Callers)
              |       $40k          |     Base larga, entrada
              +---------------------+

REGRAS:
+-- Base larga de BDRs alimenta niveis superiores
+-- Promocao baseada em metricas, nao tempo
+-- NUNCA pular niveis (BDR nao vai direto para BC)
```

### Farm System (Hormozi)

Sistema de desenvolvimento inspirado no baseball. Cada nivel treina para o proximo.

```
+------------------------------------------------------------------------------+
|  FARM SYSTEM: Desenvolvimento Progressivo                                    |
+------------------------------------------------------------------------------+
|                                                                              |
|  DOUBLE-A (BDR)                                                              |
|  +-- Cold calling, volume alto                                               |
|  +-- Ramp up: 2 DIAS (com calls gravadas)                                    |
|  +-- Metrica: 100 calls -> 20 pickups -> 2 shows                             |
|                                                                              |
|  TRIPLE-A (SDS)                                                              |
|  +-- Qualificacao, entrega de valor                                          |
|  +-- Promovido de BDR que atingiu metricas consistentemente                  |
|                                                                              |
|  BIG LEAGUES (BC)                                                            |
|  +-- Fechamento                                                              |
|  +-- Promovido de SDS que demonstrou habilidade de close                     |
|  +-- Trial: 10 deals antes de confirmar promocao                             |
|                                                                              |
|  PONTO: O SISTEMA de treinamento e um ASSET do negocio                       |
|  PONTO: Nao e custo - e vantagem competitiva duravel                         |
|                                                                              |
+------------------------------------------------------------------------------+
```

```
+--------------+--------------+--------------+
|     BDR      |     SDS      |      BC      |
+--------------+--------------+--------------+
| Jr BDR       | Jr SDS       | Jr BC        |
| Associate BDR| Associate SDS| Associate BC |
| BDR          | SDS          | BC           |
| Senior BDR   | Senior SDS   | Senior BC    |
+--------------+--------------+--------------+

REGRAS DE PROMOCAO:
+-- Promocoes a cada ~8 semanas
+-- Baseadas em metricas objetivas
+-- Titulo importa MAIS que bump salarial ($50->$55)
+-- "People lose hope when they run out of future"

STEAM DURATION:
+-- Vendedor mantem "steam" por 6-8 semanas
+-- Apos isso, precisa de "something new" (promocao, desafio)
```

## Armadilhas

### Pular Niveis

Promover BDR direto para BC pula etapa critica de desenvolvimento. SDS aprende qualificacao e entrega de valor - skills que o BC precisa.

### Ignorar Steam Duration

Vendedor mantem momentum por 6-8 semanas. Apos isso, precisa de novo desafio ou promocao. Se voce nao oferece, o concorrente oferece.

### Treinamento como Custo

Ver treinamento como despesa em vez de asset. O sistema de treinamento e vantagem competitiva - quanto mais refinado, mais rapido novos vendedores produzem.

### SDR Generico no Brasil

Colocar mesmo perfil de SDR para trabalhar lead de funil, social e cold call. Cada origem exige especializacao diferente.
