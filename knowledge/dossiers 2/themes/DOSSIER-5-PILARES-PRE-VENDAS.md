## Filosofia Central

A divisao pre-vendas/vendas (Aaron Ross) e o que grandes empresas usam para comercial eficiente e previsivel. Melhor ter pre-vendas bem estruturado que area de vendas - e o coracao da empresa. SDR nao e atendentezinho - mostra personalidade, irreverencia, nao e subordinado ao cliente.

```
+==============================================================================+
|  "Agendamento e maior KPI - prefiro conversao baixa com muitos               |
|   agendamentos que alta conversao com poucos"                                |
|                                                 -- Full Sales System         |
+==============================================================================+
|                                                                              |
|  PRINCIPIOS FUNDAMENTAIS:                                                    |
|                                                                              |
|  1. AGENDAMENTO E CONQUISTADO, NAO RECEBIDO                                  |
|     +- Se voce so agendar quem quer ser agendado, deixa dinheiro na mesa     |
|     +- Cliente dificil de agendar = mais provavel e valioso de comprar       |
|     +- Cliente dificil quando agenda, compra e compra muito                  |
|                                                                              |
|  2. LEAD SE PEGA FRESCO                                                      |
|     +- Lead se pega quente - demora = probabilidade cai                      |
|     +- 5 minutos: tempo maximo para primeiro contato (funil digital)         |
|     +- 20 minutos: tempo para contato em aplicacoes                          |
|                                                                              |
|  3. ESPECIALIZACAO E PODER                                                   |
|     +- Vendedores que fazem tudo nao fazem nenhum bem                        |
|     +- Divisao clara: SDR agenda, Closer fecha                               |
|     +- Cada um faz UMA coisa com excelencia                                  |
|                                                                              |
|  4. SORRISO SE OUVE PELO TELEFONE                                            |
|     +- Sorriso mostra que voce gosta da pessoa                               |
|     +- Seres humanos gostam de quem se interessa por eles                    |
|     +- Principios Dale Carnegie aplicados                                    |
|                                                                              |
+==============================================================================+
```

## Modelos Mentais Chave

```
+------------------------------------------------------------------------------+
|                         MODELOS MENTAIS SDR                                  |
+------------------------------------------------------------------------------+
|                                                                              |
|  1. METAFORA DA CONQUISTA (JOAO E A MOCA)                                    |
|     +- Homem querendo conquistar mulher                                      |
|     +- Ela diz nao, ele da passo atras (nao ameaca)                          |
|     +- Mostra que e legal, conquista aos poucos                              |
|     +- Agendamento = mesma dinamica                                          |
|                                                                              |
|  2. SPIN DO CLIENTE                                                          |
|     +- Humor/propensao para comprar                                          |
|     +- SDR que gera bom spin = conversao 50%                                 |
|     +- SDR comum = 15-20% conversao                                          |
|     +- Spin mensuravel via taxa de conversao por SDR                         |
|                                                                              |
|  3. PARADOXO DA ESCOLHA                                                      |
|     +- Muitas opcoes = pessoa nao escolhe                                    |
|     +- Menos opcoes = decisao mais facil                                     |
|     +- Sempre oferecer apenas 2 horarios                                     |
|                                                                              |
|  4. TEMPO VS DINHEIRO (MOEDA DO CLIENTE)                                     |
|     +- Agendamento: quer moeda TEMPO                                         |
|     +- Venda: quer moeda DINHEIRO                                            |
|     +- High ticket: tempo vale mais que dinheiro                             |
|                                                                              |
|  5. MICRO COMMITMENTS                                                        |
|     +- "Posso te passar os horarios?" -> "Sim"                               |
|     +- Ja deixou micro compromisso                                           |
|     +- Depois pergunta qual horario fica melhor                              |
|                                                                              |
|  6. PRINCIPIOS DALE CARNEGIE APLICADOS                                       |
|     +- Genuinamente interessado nas pessoas                                  |
|     +- Sorrir (mesmo pelo telefone)                                          |
|     +- Lembrar nome da pessoa                                                |
|     +- Ser bom ouvinte                                                       |
|     +- Focar no interesse do outro                                           |
|     +- Fazer pessoas se sentirem importantes                                 |
|                                                                              |
+------------------------------------------------------------------------------+
```
