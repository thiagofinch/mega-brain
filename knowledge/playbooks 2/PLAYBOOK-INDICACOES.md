## MODELO MENTAL: PEGAR vs PEDIR

```
+---------------------------------------------------------------------------------+
|  PEDIR INDICACAO (Amador)        vs     PEGAR INDICACAO (Profissional)          |
+---------------------------------------------------------------------------------+
|                                                                                 |
|  - Apelo emocional                      - Processo estruturado                  |
|  - Depende de boa vontade               - Produz resultados previsiveis         |
|  - "Voce conhece alguem?"               - 6 Microcommitments sequenciais        |
|  - Resposta: "Vou pensar..."            - Resposta: 10 nomes + telefones        |
|  - Sem follow-up                        - Validacao ao vivo                     |
|  - Resultados esporadicos               - Resultados consistentes               |
|                                                                                 |
+---------------------------------------------------------------------------------+
```

## FRAMEWORK: 6 MICROCOMMITMENTS

```
+---------------------------------------------------------------------------------+
|  PROCESSO PEGAR INDICACOES - 6 MICROCOMMITMENTS                                 |
+---------------------------------------------------------------------------------+
|                                                                                 |
|  MICROCOMMITMENT 1 - SATISFACAO                                                 |
|  +-- "Em uma escala de 0-10, quao satisfeito voce esta?"                        |
|  +-- Estabelece estado emocional positivo                                       |
|                                                                                 |
|  MICROCOMMITMENT 2 - RAZAO                                                      |
|  +-- "O que mais te deixou satisfeito?"                                         |
|  +-- Cliente verbaliza valor (auto-persuasao)                                   |
|                                                                                 |
|  MICROCOMMITMENT 3 - NOMES (1a Rodada)                                          |
|  +-- "Quem voce conhece que poderia se beneficiar?"                             |
|  +-- Meta: 5 nomes minimo                                                       |
|                                                                                 |
|  MICROCOMMITMENT 4 - MAIS NOMES (2a Rodada)                                     |
|  +-- "E alem desses, mais alguem?"                                              |
|  +-- Meta: +5 nomes = total 10                                                  |
|                                                                                 |
|  MICROCOMMITMENT 5 - TELEFONES                                                  |
|  +-- "Qual o contato de cada um?"                                               |
|  +-- So pede DEPOIS de ter os nomes                                             |
|                                                                                 |
|  MICROCOMMITMENT 6 - VALIDACAO AO VIVO                                          |
|  +-- "Vamos ligar agora para confirmar interesse?"                              |
|  +-- Accountability + proximo passo imediato                                    |
|                                                                                 |
|  Principio: Cada pequeno SIM prepara o grande SIM                               |
|  Fonte: [BATCH-126] Full Sales System - PAF-0100                                |
+---------------------------------------------------------------------------------+
```

```

+---------------------------------------------------------------------------------+
|                                                                                 |
|  QUANDO: Indicacao converte em venda                                            |
|                                                                                 |
|  EXECUCAO:                                                                      |
|  +-- Indicacao fecha: "Parabens! O [indicado] fechou!"                          |
|  +-- Convite: "Quero te agradecer. Posso te levar pra jantar?"                  |
|  +-- No jantar: Pedir mais indicacoes (cliente em estado de gratidao)           |
|                                                                                 |
|  PSICOLOGIA:                                                                    |
|  +-- Reciprocidade + Reconhecimento + Ciclo de Indicacao                        |
|  +-- Cliente se sente valorizado e quer indicar mais                            |
|  +-- Jantar vira oportunidade de networking e novas indicacoes                  |
|                                                                                 |
|  ROI:                                                                           |
|  +-- Jantar: ~R$ 200-500                                                        |
|  +-- Comissao de uma venda: ~R$ 3.000-10.000+                                   |
|  +-- ROI: 10-50x                                                                |
|                                                                                 |
```
