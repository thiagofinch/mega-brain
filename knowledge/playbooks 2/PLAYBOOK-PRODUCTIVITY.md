## PTB - Prospect Time Blocking Framework

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  PTB - PROSPECT TIME BLOCKING (NEPQ PRODUCTIVITY)                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  PRINCÍPIO: "What am I doing RIGHT NOW? Is it producing sales or income?"   │
│                                                                             │
│  SCHEDULE DE $500K+/ANO EM COMISSÕES:                                       │
│  ├── 05:00 → Acordar                                                        │
│  ├── 05:30-06:30 → Academia (max 15min do gym/trabalho)                    │
│  ├── 07:30 → PRIMEIRO check de emails                                      │
│  ├── 08:00-11:00 → PTB (Cold calling, outbound, buyer leads)               │
│  │                 Mente mais afiada = tarefas mais difíceis               │
│  ├── 11:00-13:00 → Booked appointments                                     │
│  ├── 13:00-13:20 → Almoço (20 MIN, não 1 hora!)                           │
│  │                 SEGUNDO check de emails enquanto come                   │
│  ├── 13:30-17:30 → Mais appointments + follow-up se no-show               │
│  ├── 17:30 → Sair do escritório                                            │
│  └── 22:00 → TERCEIRO e último check de emails                             │
│                                                                             │
│  REGRA DE EMAIL: Apenas 3x por dia. NUNCA a cada 10-30 minutos.            │
│                                                                             │
│  Fonte: [BATCH-062] Jeremy Miner - JM-0102, JM-0104                        │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Income-Producing vs Non-Income Activities

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  INCOME-PRODUCING ACTIVITIES (IPAs):                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ✅ Cold calling prospects                                                  │
│  ✅ Calling outbound leads / buyer leads                                    │
│  ✅ Booked appointments                                                     │
│  ✅ Follow-up com leads que não compraram                                   │
│  ✅ Esperar email de prospect prestes a pagar                              │
│                                                                             │
│  NÃO É INCOME-PRODUCING:                                                    │
│  ❌ Checar emails constantemente                                            │
│  ❌ Almoços longos (1 hora)                                                 │
│  ❌ Ficar esperando quando no-show                                          │
│  ❌ "Company Club" - ficar de papo com vendedores negativos                │
│                                                                             │
│  REGRA: Se appointment não aparecer = imediatamente follow-up ou calling   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
