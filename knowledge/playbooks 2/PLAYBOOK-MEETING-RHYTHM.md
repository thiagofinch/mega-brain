## FRAMEWORK MASTER: Meeting Rhythm Architecture

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    MEETING RHYTHM ARCHITECTURE                              │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  QUARTERLY:                                                                 │
│  ├── QSP (Quarterly Sprint Plan) - 2nd week last month of quarter          │
│  │   └── Leadership team | Full day (first couple) → Eventually 4hrs       │
│  │   └── ABSORBS MBR in that month                                         │
│  └── Team Day/All-Hands - 3rd week (after QSP, after financials close)     │
│      └── Everyone | Share targets, celebrate, distribute profit shares     │
│                                                                             │
│  MONTHLY:                                                                   │
│  └── MBR (Monthly Business Review) - Last Monday of month                  │
│      └── Leadership team | 2 hours max                                     │
│      └── ABSORBS weekly leadership in that week                            │
│                                                                             │
│  WEEKLY:                                                                    │
│  ├── Leadership Meeting - Mondays | 1 hour                                 │
│  │   └── Review company scorecard                                          │
│  └── Team Meetings/Safety - Tuesdays (day after leadership)                │
│      └── Team scorecard review                                             │
│                                                                             │
│  AD-HOC:                                                                    │
│  └── One-on-ones - Field/as needed | 30 min max                            │
│      └── Development, temperature check, resource needs                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## ABSORPTION MODEL

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    MEETING ABSORPTION MODEL                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  PRINCIPLE: Higher-level meetings CONTAIN lower-level meetings             │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  QUARTERLY: QSP                                                      │   │
│  │  └── Contains: MBR content + QSP-specific planning                  │   │
│  │  └── When QSP happens, MBR is ABSORBED (not held separately)        │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                               ↓                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  MONTHLY: MBR                                                        │   │
│  │  └── Contains: Weekly review content + MBR-specific reviews         │   │
│  │  └── When MBR happens, weekly leadership is ABSORBED                │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                               ↓                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  WEEKLY: Leadership + Team                                           │   │
│  │  └── Contains: Scorecard review + projects + bottlenecks            │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  BENEFIT: Avoids meeting overload, maintains rhythm without redundancy     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## WEEKLY CHECK-IN STRUCTURE

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    WEEKLY CHECK-IN STRUCTURE                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  DURATION: 60-90 minutes MAX (shorter is better)                           │
│  CADENCE: Every Monday for leadership, Tuesday for teams                   │
│                                                                             │
│  5-PART AGENDA:                                                             │
│                                                                             │
│  1. WINS (5 min)                                                           │
│     └── Each person shares 1 win from past week                            │
│     └── Sets positive tone                                                 │
│                                                                             │
│  2. SCORECARD REVIEW (15-20 min)                                           │
│     └── Review traffic light colors                                        │
│     └── Red/Yellow items get attention                                     │
│                                                                             │
│  3. PROJECT UPDATES (20-30 min)                                            │
│     └── Status on quarterly rocks/projects                                 │
│     └── On-track vs off-track                                              │
│                                                                             │
│  4. BIG THREE REPORTING (10 min)                                           │
│     └── Each person states 3 highest-priority tasks for next week          │
│     └── Creates accountability and visibility                              │
│                                                                             │
│  5. BOTTLENECKS (10-15 min)                                                │
│     └── Surface and solve blockers                                         │
│     └── Assign owners and deadlines                                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## QUARTERLY SPRINT PLANNING (QSP)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    QSP METHODOLOGY                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  WHEN: 2nd week of last month of quarter                                   │
│  WHO: Leadership team                                                       │
│  DURATION: Full day (first few) → 4 hours (trained team)                   │
│                                                                             │
│  FIRST QSP EVER (2-Day):                                                   │
│  ├── Day 1: CLARITY DAY                                                    │
│  │   ├── Finalize Clarity Compass WITH team                                │
│  │   ├── 3-year target                                                     │
│  │   ├── Company purpose                                                   │
│  │   ├── Core values                                                       │
│  │   └── Strategic anchors                                                 │
│  │   └── NOTE: Don't repeat for 3 YEARS                                    │
│  │                                                                          │
│  └── Day 2: QUARTERLY SPRINT PLANNING                                      │
│      └── Build 90-day plan using 12Q grid                                  │
│                                                                             │
│  SUBSEQUENT QSPs:                                                           │
│  └── 4-8 hours for quarterly planning only                                 │
│  └── Review previous quarter                                               │
│  └── Set next 90-day rocks                                                 │
│  └── Update scorecard targets                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## MEETING SCHEDULE METHODOLOGY

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    CONCRETE SCHEDULING                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  EXAMPLE FOR Q2 (April-June):                                              │
│                                                                             │
│  JUNE (End of Quarter):                                                    │
│  ├── Week 2 (Mon): QSP for Q3 planning                                     │
│  │   └── e.g., June 10 (Monday) - Leadership                               │
│  │   └── Absorbs MBR for June                                              │
│  │   └── Absorbs weekly leadership for that week                           │
│  │                                                                          │
│  └── Week 3 (Fri): Team Day/All-Hands                                      │
│      └── e.g., June 20 (Friday) - after financials close                   │
│      └── Share Q3 targets, celebrate Q2                                    │
│                                                                             │
│  JULY (Start of New Quarter):                                              │
│  ├── Every Monday: Weekly leadership (1 hr)                                │
│  ├── Every Tuesday: Team meetings                                          │
│  └── Last Monday (July 29): MBR for July                                   │
│      └── Absorbs that week's leadership meeting                            │
│                                                                             │
│  AD-HOC:                                                                   │
│  └── One-on-ones: As needed, field-based, 30 min max                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## FILOSOFIAS CORE

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    MEETING PHILOSOPHY                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1. "52 pivot opportunities per year"                                       │
│     └── Weekly meetings = 52 chances to course-correct                     │
│     └── vs entrepreneurs who check in 1-2x per year                        │
│                                                                             │
│  2. "Airplane navigation"                                                   │
│     └── Planes are almost always off-course                                │
│     └── Constant small corrections = arrival at destination                │
│     └── Weekly meetings are your navigation system                         │
│                                                                             │
│  3. "Absorb, don't add"                                                    │
│     └── Higher meetings contain lower meetings                             │
│     └── Never create meeting overload                                      │
│                                                                             │
│  4. "Rhythm over frequency"                                                │
│     └── Consistent cadence matters more than more meetings                 │
│     └── Teams can plan around predictable rhythm                           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## ANTI-PATTERNS

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    MEETING ANTI-PATTERNS                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ❌ ANNUAL PLANNING ONLY                                                   │
│     └── "Most annual plans abandoned by February" (like NY resolutions)    │
│     └── Too long to predict, too short to achieve                          │
│                                                                             │
│  ❌ MONTHLY REVIEWS ONLY                                                   │
│     └── Too slow to course-correct                                         │
│     └── Problems fester for weeks                                          │
│                                                                             │
│  ❌ NO RHYTHM                                                              │
│     └── Ad-hoc meetings create chaos                                       │
│     └── Team can't plan their work                                         │
│                                                                             │
│  ❌ MEETING OVERLOAD                                                       │
│     └── Adding more meetings instead of absorbing                          │
│     └── Creates context-switching tax                                      │
│                                                                             │
│  ❌ STATUS UPDATES ONLY                                                    │
│     └── Meetings become one-way information dumps                          │
│     └── No problem-solving, no accountability                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
