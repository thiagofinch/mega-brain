# Knowledge

Sua base de conhecimento. Gerada automaticamente pelo pipeline de processamento.

## Subpastas
- `dossiers/persons/` - Dossiês compilados por especialista
- `dossiers/themes/` - Dossiês temáticos (vendas, pricing, etc.)
- `dossiers/system/` - Dossiês do sistema
- `sources/` - Índices de fontes processadas
- `playbooks/` - Playbooks acionáveis gerados
- `dna/` - DNAs cognitivos extraídos (5 camadas)
